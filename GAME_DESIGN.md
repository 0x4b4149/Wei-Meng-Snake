# 威威孟孟專屬：純愛貪食蛇 (Wei & Meng Snake) 遊戲設計文件

## 1. 遊戲簡介
這是一款基於 Vue 3 開發的網頁版貪食蛇遊戲，主題環繞著知名同志情侶網紅「崴寶」與「孟寶」。遊戲結合了他們的迷因特色、雙方專長（AI 與音樂），以及極度純真的純愛氛圍，打造出一款充滿甜蜜與驚喜的趣味小遊戲。

## 2. 核心玩法與機制
傳統貪食蛇玩法結合特殊主題元素：
*   **控制方式**：崴寶使用鍵盤方向鍵 (上、下、左、右)，孟寶使用(W、A、S、D)。
*   **勝利條件**：無盡模式，目標是取得最高分數。
*   **失敗條件**：撞到牆壁、撞到自己的身體、碰到彼此的身體。

## 3. 主題特色元素設計

### 3.1 角色設定 (貪食蛇)
*   **蛇頭**：玩家有兩名，分別為「崴寶」或「孟寶」，並以個別的大頭貼作為蛇頭，崴寶請使用 崴寶的圖片[./wei.png] 孟寶請使用 孟寶的圖片[./meng.jpg]
*   **蛇身**：隨著吃到食物，身體會變長，身體的圖案會是滿滿的愛心，或是兩人的 Q 版牽手動畫。

### 3.2 食物與道具 (迷因與專長結合)
*   **普通食物：草莓蛋糕 🍰**
    *   **效果**：長度 +1，分數 +10。
    *   **彩蛋**：吃到時，畫面會隨機飄出文字「**它的顆粒好好吃！**」，並伴隨孟寶真誠讚嘆的音效。
*   **特殊道具 1：AI 晶片 💻 (崴寶專屬)**
    *   **效果**：短暫開啟「自動導航」模式（約 5 秒），蛇會自動尋找並吃掉畫面上的食物。象徵崴寶的 AI 專長。
*   **特殊道具 2：鋼琴鍵盤 🎹 (孟寶專屬)**
    *   **效果**：吃到後播放一段輕快的即興音樂，並啟動「慢動作」模式，讓玩家更容易閃避危險。象徵孟寶的音樂才華。
*   **特殊道具 3：三百天禮物 🎁**
    *   **效果**：分數大幅增加，並在畫面上顯示一段「崴孟繞口令」，玩家若能在時間內不撞牆，分數加倍。

### 3.3 視覺與聽覺設計
*   **遊戲背景**：以粉色系、柔和的暖色調為主，營造純愛與熱戀的氛圍。
*   **背景音樂 (BGM)**：採用他們的原創歌曲（例如：〈崴孟三百天禮物〉的 8-bit 版本或輕音樂版）。
*   **音效**：除了吃蛋糕的迷因音效，撞牆失敗時會有兩人互相安慰的語音（例如：「沒關係，下次再努力！」）。

## 4. Vue 技術架構建議
*   **框架**：Vue 3 + Composition API
*   **建構工具**：Vite (已內建於當前專案)
*   **狀態管理**：使用 `ref` 與 `reactive` 管理蛇的座標、食物座標、分數與遊戲狀態（開始、暫停、結束）。
*   **渲染方式**：推薦使用 CSS Grid 建立遊戲網格，或使用 HTML5 `<canvas>` 結合 Vue 的生命週期來繪製，以獲得更好的效能。
